# 🎲 Ludo Game

This is a simple console-based Ludo game I built using Python as a practice project.

## Features

- Four-player gameplay
- Dice rolling
- Token movement
- Turn-based gameplay
- Win condition

## Built With

- Python

## How to Run

1. Clone this repository.
2. Open the project in any Python IDE (or VS Code).
3. Run the Python file.

## What I Learned

- Writing game logic
- Using functions effectively
- Working with loops and conditions
- Debugging and fixing errors
